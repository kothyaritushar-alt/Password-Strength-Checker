# Security Policy

## Supported Versions

This project is a small educational security tool. Only the latest version in the `main` branch is actively maintained.

---

## Purpose & Scope

This repository contains a **defensive password strength checker** designed for learning, demonstration, and portfolio purposes.

- It does **not** crack passwords
- It does **not** store passwords
- It does **not** transmit passwords
- It performs **local analysis only**

---

## Responsible Usage Guidelines

Please follow these rules when using or extending this tool:

- ✅ Use only **test passwords** or passwords you own
- ❌ Do NOT analyze real user passwords
- ❌ Do NOT log, store, or upload passwords
- ❌ Do NOT integrate this tool into systems without proper security review

This tool is intended to demonstrate **password hygiene concepts**, not to bypass authentication mechanisms.
